# AI Grocery Assistant Chatbot
AI-powered chatbot to help with grocery shopping. Built with **Python**, **Streamlit**, and **n8n** for workflow automation.

## Features

- Suggest groceries based on user preferences.
- Automated workflow using n8n for integrations and notifications.
- Simple and interactive interface using Streamlit.

## Tech Stack

- **Python** – Core logic and UI ( streamlit )
- **Streamlit** – Frontend interface for user interaction
- **n8n** – Workflow automation and task scheduling
- **Google Gemini API / GPT** – AI chatbot responses
